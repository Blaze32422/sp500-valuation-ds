# S&P 500 Valuation Analysis
A data science project to rank S&P 500 companies by value per EPS, using pandas for data processing, matplotlib/seaborn for visualizations, and Streamlit for an interactive web app. Includes exploratory data analysis (EDA) and a downloadable HTML report.

## Features
- Calculates a value score based on Price-to-EPS ratio, revenue growth, and market cap.
- Performs EDA to analyze sector distributions and financial metrics.
- Visualizes results with interactive bar and scatter plots.
- Generates a professional HTML report with embedded plots.
